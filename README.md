# Kaushik Plabon - Magisk Modules

A collection of experimental Magisk modules focused on system behavior tuning, smoothness, and battery efficiency.

## Modules

### 1. Authentic Dynamic Refresh Rate
Dynamic refresh rate control based on usage:
- Idle 30 Hz
- Scrolling 120 Hz
- Most Apps 90 Hz
- Video playback 60-90 Hz
- Reading 30 Hz

### 2. Dynamic Animations
Adjusts animation speed based on current refresh rate for a smoother UI experience.

---

## Installation

1. Download the module ZIP from Releases
2. Open Magisk
3. Go to Modules
4. Install from storage
5. Reboot
